Set a simple EmBZ treatment scenario

This guide explains how to use Matlab to quickly define a single EmBZ treatment model scenario

Setting a single treatment

The NewDepomod.InputPropertiesFile class has some convenience methods for easily setting up simple discharge scenarios.
For setting a single EmBZ treatment the .setEmBZQuantity() method can be used. This method should be used in conjunction with the .setBiomass() or .setStockingDensity() methods.
The .setEmBZQuantity() method takes a treatment quantity in g and populates columns 3 and 6 of the inputs data table discharges time series. These columns refer to the waste and faecal medicine residue discharge quantities. Note that column 1, 2, 4 and 5 - which represent organic solids and carbon discharges are unaffected by this method and therefore should be set separately.
The typical pattern for a standard EmBZ treatment would be:
inputs = run.inputsFile;
inputs.setBiomass(1000, 'days', 118);
inputs.setEmBZQuantity(500, 'days', 118);
This combination of methods sets up the inputs file to simulate, in this case, a constant biomass of 1000 t and a treatment of 500 g over a simulation of 118 days. The inputs file data table is populated appropriately, as are the pertinent header properties.
Strictly speaking, the 'days' argument is not required in this case for the .setEmBZQuantity() method as 118 days is the default length, but it is good practice to declare it explicitly.
Note, that the .setBiomass() method completely clears the discharge time series data table and therefore must be called before .setEmBZQuantity().
This function makes an assumption about the typical feed wastage rate which can be overridden using the 'feedWastePercentage' option.

Writing to file

Any changes made to the inputs properties file using this method must be explicitly written back to file in order for them to take effect in the software.
