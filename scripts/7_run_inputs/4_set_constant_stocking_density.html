Set a constant stocking density scenario

This guide explains how to use Matlab to quickly define a constant stocking model scenario

Setting a constant stocking density

The NewDepomod.InputPropertiesFile class as some convenience methods for easily setting up simple discharge scenarios.
For setting a constant level of discharge the .setStockingDensity() method can be used. This is very similar to the .setBiomass() but operates according to a specific stocking density rather than absolute biomass.
inputs = run.inputsFile;
inputs.setBiomass(19.5);
This method sets the FeedInputs.biomass and FeedInputs.stockingDensity properties in the header of the inputs file, and derives the hourly discharge data equivalent to the specified biomass and populates the inputs file data table.
This function makes a number of assumptions about the discharge scenario, such as 365 day discharge duration, a constant feed rate of 7 kg feed per t biomass per day, and a variety of conversion factors for feed wastage rate, water content, digesibility, etc. in order to convert the notional feed load into waste feed and faecal discharge rates.
These assumptions can be overridden by using some or all of the following arguments:
  • 'days' - specify the number of simulation days
  • 'feedRatio' - the feed rate (kg/t/d)
  • 'feedWaterPercentage'
  • 'feedWastePercentage'
  • 'feedAbsorbedPercentage'
  • 'feedCarbonPercentage'
  • 'faecesCarbonPercentage'
For example,
inputs.setStockingDensity(19.5, 'days', 90, 'feedRatio', 8)
Note that, in the case of specifying a non-default number of days, the inputs file data table is set to have the appropriate number of rows and the FeedInputs.numberOfTimeSteps property is set correspondingly.

Writing to file

Any changes made to the inputs properties file using this method must be explicitly written back to file in order for them to take effect in the software.
inputs.toFile
