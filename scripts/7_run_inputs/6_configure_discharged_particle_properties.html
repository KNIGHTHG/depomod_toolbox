Configure discharged particle properties

This guide explains how to use Matlab to set the settling velocities of the discharged particles

Note

The inputs properties file describes the characteristics of the discharged particles. This include the characteristics of waste feed and faecal particles. A particularly useful property with respect to these particles are their settling velocity. This guide will explain how to set these values.
A third type of particle which is represented in NewDepomod is a resuspended "mud" particle, which also has a settling velocity property. These properties are configured using the run physical properties file. The remit of the inputs property file covers particles discharged from the farm cages.

Understanding settling velocities

Waste feed and faeces settling velocities are specified using the following properties
inputs = run.inputsFile;
inputs.Particle.settlingVelocityOfFaeces
ans = struct with fields:
dispersion: '0.003500'
distribution: 'UNIFORM'
location: '-0.035000'
inputs.Particle.settlingVelocityOfFeed
ans = struct with fields:
dispersion: '0.009500'
distribution: 'UNIFORM'
location: '-0.095000'
Notice they both have 3 components to their definition. This is because settling velocities assigned to particles in the model are able to be sampled from a distribution rather than necessarily being constant across all particles.
The 3 components represent the distribution type (.distribution), the distribution magnitude (.location) and the distribution spread (.dispersion). In the example above the settling velocities are defined using uniform distributions. This means that there is an equal probability of any value across the specified range. In the case of the faeces particles, the settling velocity is specified to have a mean value of 3.5 cm/s and a deviation of +/- 3.5 mm/s.

Setting the settling velocity

An example of changing these values is given below
inputs.Particle.settlingVelocityOfFaeces.distribution = 'GAUSSIAN';
inputs.Particle.settlingVelocityOfFaeces.location = num2str(-0.032);
inputs.Particle.settlingVelocityOfFaeces.dispersion = num2str(0.0032);
inputs.Particle.settlingVelocityOfFaeces
ans = struct with fields:
dispersion: '0.0032'
distribution: 'GAUSSIAN'
location: '-0.032'
This defines a normally distributed settling velocity. Note that the settling velocity magnitude should be specified as a negative number indicating a downwards velocity.
These changes must be written to file in order for them to take effect in the software.
