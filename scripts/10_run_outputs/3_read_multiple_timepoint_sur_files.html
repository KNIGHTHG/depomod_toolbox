Reading in multiple sur files

In order to read in multiple files we need the timestamps at which they were outputted. These can be retrieved easily using
t = run.outputTimestamps
t = 1×29
86400 172800 259200 345600 432000 518400 604800 691200 777600 864000 950400 1036800 1123200 1209600 1296000 1382400 1468800 1555200 1641600 1728000 1814400 1900800 1987200 2073600 2160000 2246400 2332800 2419200 2505600
These are the output times (in seconds) for which we should have sur files. There are 29 of them. We can read all of them in like this:
surs = run.sur('type', 'solids', 'g', 0, 't', t)

Interogating multiple surs

Now we have all the surs instantiated we can query them. For example, let's produce a time series of masses at a single point.
masses = [];
for ts = 1:numel(surs)
sur = surs{ts};
mass = sur.valueAt(350427,1069122);
masses(ts) = mass;
end
figure;
plot(t, masses, 'ro')
xlabel('time (seconds)')
ylabel('mass at location (g m^{-2})')
grid on
Beautiful.
